## 项目概述

简洁直观的平台，用户可以创建班级、发出作业、批改作业并管理个人账户。

---

## 运行项目

### 安装步骤

1. **克隆仓库**：
   ```bash
   git clone https://github.com/fjc663/cstery-lms.git
   ```

2. **安装依赖**：
   进入项目目录并运行：
   ```bash
   npm install
   ```

3. **运行开发服务器**：
   ```bash
   npm run dev
   ```
   用户端和管理端将在本地运行。端口为5173

4. **访问**

   访问项目: [http://localhost:5173/](http://localhost:5173/)
